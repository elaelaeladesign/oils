# Essential Oils – doTERRA Guide

A simple static, responsive webpage showcasing 8 doTERRA essential oils.

## How to publish on GitHub Pages
1. Create a new repository (e.g. `essential-oils-guide`)
2. Upload all files (including the `images` folder)
3. Go to **Settings → Pages → Source → main branch**
4. Wait 1–2 minutes and open your page:
   ```
   https://<your-username>.github.io/essential-oils-guide/
   ```
